
# Sistema de Pedidos de Farmácia – Design Patterns em Java

## Descrição do Projeto
Simula um sistema de pedidos para farmácia aplicando Singleton, Strategy e Facade.

## Padrões Aplicados
- **Singleton:** BancoDados – instância única para armazenar pedidos.
- **Strategy:** PagamentoStrategy – Cartão, Boleto, Pix.
- **Facade:** PedidoFacade – interface simples para criar e finalizar pedidos.

## Como Executar
1. Compile:
javac */*.java Main.java
2. Execute:
java Main
